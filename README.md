# 🌍 AI Language Translation App

An AI-powered multilingual language translation application built using Streamlit.

## Features

* Translate between multiple languages
* Source and target language selection
* Text-to-Speech support
* Translation history
* Download translated output
* Character counter

## Technologies

* Python
* Streamlit
* Deep Translator
* Google Text To Speech

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
